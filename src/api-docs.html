API Documentation

Complete reference for E-Clean REST API. Base URL: https://api.e-clean.com/v1

🔐 Authentication

POST /auth/login

Authenticate user and get access token

email string User email address
password string User password
Request
curl -X POST https://api.e-clean.com/v1/auth/login \ -H "Content-Type: application/json" \ -d '{"email": "user@example.com", "password": "secret"}'
200 Success 401 Invalid credentials
POST /auth/register

Register new user account

email string User email (unique)
password string Min 8 characters
first_name string First name
last_name string Last name

🛍️ Products

GET /products

List all products with filtering and pagination

category string Filter by category slug
search string Search in name/description
min_price number Minimum price filter
max_price number Maximum price filter
page integer Page number (default: 1)
limit integer Items per page (max: 50)
Response
{ "success": true, "data": [...], "pagination": { "page": 1, "limit": 20, "total": 150, "pages": 8 } }
GET /products/:id

Get single product details

GET /products/:id/recommendations

Get AI-powered product recommendations

📦 Orders

GET /orders

List user orders

POST /orders

Create new order

items array Array of {product_id, quantity}
shipping_address object Shipping address object
payment_method string "card" | "paypal"
coupon_code string Optional discount code
GET /orders/:id/tracking

Get order tracking information

💳 Payments

POST /payments/create-intent

Create Stripe payment intent

POST /payments/confirm

Confirm payment with card details

🎫 Support

POST /support/tickets

Create support ticket

GET /support/tickets/:id

Get ticket details and messages

🤖 AI Endpoints

POST /ai/chat

Chat with E-Clean AI Assistant

message string User message
session_id string Session identifier
language string Response language (fr/en/es)
Example Response
{ "success": true, "response": "Je vous recommande le nettoyant multi-surfaces Pro...", "agent": "sales", "action": "recommendation", "timestamp": "2026-04-20T12:00:00Z" }
GET /ai/analytics

Get AI system analytics
